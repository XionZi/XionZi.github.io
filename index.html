Blog维护参考文档

发表于 2020-04-15

命令

/site 目录下

  • hexo new post “blog”

    生成 blog.md

  • hexo g (=hexo generate)

    开始生成Hexo

  • hexo s (=hexo server)

    开启本地服务，默认端口4000

  • hexo server -p {port}

    开启本地服务

  • hexo d -g

    快速生成和部署远程仓库

  • hexo clean

    清除缓存，可在切换主题之后、验证之前使用

访问

  • 本地

    localhost:4000

  • github.io

    https://XionZi.github.io
